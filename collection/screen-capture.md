Screen Capture
=============
Malware takes screen captures of the desktop.

**See ATT&CK:** [**Screen Capture**](https://attack.mitre.org/techniques/T1113/).

Methods
-------
|ID|Name|Description|
|-----------------------------|--------|-----------------------------|
|E1113.m01|**WinAPI**|Screen is captured using WinAPI functions (e.g., user32.GetDesktopWindow).|
